## Features
**1️⃣ 언제 복용해야하더라**
카테고리별로 알람시간을 지정하여 복약 알림을 받을 수 있습니다

**2️⃣ 오버도즈는 위험해**
복용해야하는 용량이 넘어가면 눈에 띄는 색으로 오남용을 표시해줍니다

**3️⃣ 복약 기록을 모아볼까**
달력을 보듯 한눈에 한 달 동안의 복약 기록을 모아볼 수 있습니다

**4️⃣ 주간 정리가 필요해**
주간 기록을 눈에 띄는 색으로 모아볼 수 있습니다

**5️⃣ 앱에 들어가지 않아도 한눈에**
일일, 주간, 월간 위젯을 통해 복약 기록 등을 확인할 수 있습니다
